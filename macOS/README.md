# Mac/Windows 開發環境一鍵配置

本專案用於快速安裝 Node.js, Python, Git 與 VS Code。

### 🚀 快速開始 (macOS)
打開終端機並貼上以下指令：
\`\`\`bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/你的帳號/mac-init/main/core_setup.sh)"
\`\`\`

### 🚀 快速開始 (Windows)
以管理員身分打開 PowerShell 並貼上以下指令：
\`\`\`powershell
Set-ExecutionPolicy Bypass -Scope Process -Force; iwr -useb https://raw.githubusercontent.com/你的帳號/mac-init/main/setup_windows.ps1 | iex
\`\`\`
